Project Description:

-Background: The Mars Curiosity rover has several cameras and scientific instruments. Two images are selected at random from NASA's website and stitched together to give a panoramic view (https://mars.nasa.gov/msl/multimedia/raw-images/?order=sol+desc%2Cinstrument_sort+asc%2Csample_type_sort+asc%2C+date_taken+desc&per_page=50&page=0&mission=msl)

-Steps:
1.Import and View the Images
2.Register the Two Images
3.Initialize the Template
4.Initialize the panorama template
5.Create the Panorama:
	i)Create an Alpha Blender object
	ii)Create the spatial referencing object
	iii)Warp the images
	iv)Place the warped images into the panorama
